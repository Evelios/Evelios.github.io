# Intro

This concept is know by many names. You can see it called *easing function*,
*filter functions*, *lerping function* (linear interpolation functions) or
*tweening functions*.

* **easing function** - artist or animation context used for describing the motion
  of objects in movement
* **filter function** - running a filter on the input in a way to modity the input
  values to transform it into something else
* **lerping function** - linear interpolation functions which have a misleading
  name due to the fact that most easing functions are not linear
* **tweening function** - from animation standing for inbetween or creating
  animation inbetween key frames
